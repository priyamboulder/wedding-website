# The Comprehensive Indian Wedding Planning Checklist

*A granular, event-by-event, decision-by-decision reference covering every task from engagement through post-wedding. Designed to map cleanly to Ananya's module structure: Venue, Transportation, Mehndi, Stationery, Entertainment, Guest Management, Master Timeline, and Outfits & Styling.*

---

## Phase 0: Foundation & Vision (12–18 Months Before)

### Couple Alignment
- [ ] Discuss overall wedding vision (traditional / modern fusion / destination / intimate)
- [ ] Align on core values: scale, formality, religious observance, cultural fidelity
- [ ] Decide how many events total (2-day, 3-day, 4-day, 5-day)
- [ ] Agree on which side's traditions lead for each ceremony (if interfaith/inter-regional)
- [ ] Decide on tradition profile (North Indian Hindu, Punjabi Sikh, Gujarati, Marwari, Tamil Brahmin, Telugu, Malayali, Bengali, Kashmiri, Sindhi, Jain, interfaith, etc.)
- [ ] Define "non-negotiables" list for each partner
- [ ] Decide whether to have a wedding planner, day-of coordinator, or self-manage

### Family Coordination
- [ ] Formal introduction between families (if not done)
- [ ] Roka ceremony (if applicable)
- [ ] Align both sets of parents on overall plan
- [ ] Identify key decision-makers on each side
- [ ] Establish communication cadence (WhatsApp group? Weekly calls?)
- [ ] Assign family "point people" for specific domains (catering uncle, logistics cousin, etc.)
- [ ] Discuss financial contribution structure between families
- [ ] Document any family traditions that must be honored

### Budget Architecture
- [ ] Set total budget ceiling
- [ ] Allocate budget by event (typical split: Wedding 40%, Reception 20%, Sangeet 15%, Mehndi 10%, Haldi 5%, Other 10%)
- [ ] Allocate by category (venue, catering, decor, attire, jewelry, photo/video, music, transport, stationery, gifts, misc)
- [ ] Build 10–15% contingency buffer
- [ ] Decide payment responsibility by line item (bride's family, groom's family, couple, split)
- [ ] Set up shared budget tracker
- [ ] Open dedicated wedding bank account or credit card for points/rewards

### Date & Muhurat
- [ ] Consult family priest/panditji with both partners' birth details (janam kundli)
- [ ] Receive list of auspicious dates (muhurat)
- [ ] Cross-check dates against venue availability
- [ ] Cross-check against key guest availability (grandparents, close family)
- [ ] Confirm final dates for every event
- [ ] Note any religious fasting days or inauspicious periods to avoid (Shraddh, Chaturmas, Kharmas)
- [ ] Document exact muhurat times for ceremony moments (pheras, kanyadaan)

## Phase 1: Branding & Identity (10–12 Months Before)

### Wedding Brand Development
- [ ] Choose wedding hashtag (brainstorm 10+ options, test for uniqueness on Instagram)
- [ ] Check hashtag availability across Instagram, TikTok, Facebook
- [ ] Secondary hashtags per event (#PoojaAndRajSangeet, #PoojaAndRajMehndi)
- [ ] Choose wedding monogram/logo (initials, motif, mandala)
- [ ] Pick primary color palette (3–5 colors across events)
- [ ] Pick secondary/accent palette
- [ ] Choose typography system (display font + body font)
- [ ] Develop visual motif library (peacock, lotus, elephant, paisley, florals)
- [ ] Create brand guidelines doc for vendors to reference
- [ ] Decide on per-event color themes (e.g., Mehndi: mustard/coral, Sangeet: emerald/gold, Wedding: red/ivory)
- [ ] Define "wedding voice" for all written communication (formal / warm / playful)

### Digital Presence
- [ ] Register wedding website domain
- [ ] Choose wedding website platform (custom / Zola / The Knot / Joy / WedMeGood)
- [ ] Draft "Our Story" narrative
- [ ] Write how-we-met timeline
- [ ] Plan photoshoot for website hero imagery
- [ ] Create FAQ section (dress code, kids policy, alcohol, parking)
- [ ] Build event schedule page
- [ ] Build travel & accommodation page
- [ ] Build registry links page
- [ ] Set up RSVP system (per-event, per-guest, meal preferences)
- [ ] Build password protection if desired
- [ ] Plan couple's Instagram account or private sharing space
- [ ] Set up shared Google Drive / Dropbox for planning docs

## Phase 2: Core Bookings (9–12 Months Before)

### Venue Selection
- [ ] Define venue requirements per event (capacity, indoor/outdoor, catering flexibility, parking, accommodation proximity)
- [ ] Research venues for wedding ceremony
- [ ] Research venues for sangeet
- [ ] Research venues for mehndi
- [ ] Research venues for haldi
- [ ] Research venues for reception
- [ ] Research venues for welcome dinner / rehearsal
- [ ] Visit top 3 venues per event in person
- [ ] Check if venues allow outside caterers (critical for Indian food)
- [ ] Check if venues allow fire (havan/agni for ceremony)
- [ ] Check noise ordinances and end times
- [ ] Check for bridal suite / getting-ready rooms
- [ ] Confirm parking and valet capacity
- [ ] Confirm ADA accessibility
- [ ] Review insurance requirements
- [ ] Sign contracts and pay deposits
- [ ] Add all venue contacts to master vendor sheet

### Accommodation Planning
- [ ] Identify primary host hotel
- [ ] Identify overflow hotels at multiple price points
- [ ] Negotiate room block (typically 10+ rooms)
- [ ] Confirm room block cutoff date
- [ ] Set up booking link / code for guests
- [ ] Reserve hospitality suite for welcome gifts
- [ ] Book suite for bride and groom's wedding night
- [ ] Book rooms for immediate family
- [ ] Reserve getting-ready rooms
- [ ] Arrange airport shuttle / transport from hotel

### Priest & Religious Officiant
- [ ] Book panditji / priest / granthi / kazi
- [ ] Confirm language preference for ceremony (Sanskrit, Hindi, English explanations)
- [ ] Request list of puja samagri items needed
- [ ] Discuss ceremony duration and abbreviation options
- [ ] Request a written explanation of each ritual for the program
- [ ] Confirm priest's attire and expectations
- [ ] Discuss dakshina (priest's offering) amount
- [ ] Book secondary priests if multiple ceremonies

### Photography & Videography
- [ ] Research photographers with Indian wedding experience
- [ ] Review full weddings (not just highlight reels)
- [ ] Confirm number of shooters needed per event
- [ ] Book photographer
- [ ] Book videographer (often a separate vendor)
- [ ] Book cinematographer for teaser/highlight reel
- [ ] Discuss drone footage
- [ ] Book a same-day-edit vendor for sangeet/reception
- [ ] Plan engagement / pre-wedding photoshoot
- [ ] Plan save-the-date photoshoot
- [ ] Discuss photo delivery timeline and deliverables (album, USB, online gallery)
- [ ] Review contract for raw files ownership
- [ ] Book a photo booth or 360 booth for sangeet/reception

## Phase 3: Attire & Styling (6–12 Months Before)

### Bride's Wardrobe
- [ ] Define bridal aesthetic per event
- [ ] Research designers (Sabyasachi, Manish Malhotra, Anita Dongre, Tarun Tahiliani, local designers)
- [ ] Set bridal attire budget
- [ ] **Wedding lehenga**: choose color, fabric, embroidery style, silhouette
- [ ] Order custom lehenga (allow 4–6 months)
- [ ] Schedule fittings (typically 3–4)
- [ ] **Reception outfit**: gown, saree, or modern lehenga
- [ ] **Sangeet outfit**: often more playful, contemporary
- [ ] **Mehndi outfit**: yellow/green tones, comfortable for sitting
- [ ] **Haldi outfit**: yellow, old/designated-to-stain clothes
- [ ] **Welcome dinner outfit**
- [ ] **Rehearsal / puja outfit**
- [ ] **Post-wedding lunch outfit**
- [ ] Backup/second wedding outfit for after ceremony
- [ ] Undergarments and shapewear for each outfit
- [ ] Petticoats and blouses
- [ ] Coordinating dupattas and veils

### Bride's Jewelry
- [ ] Decide on jewelry style (temple, kundan, polki, meenakari, diamond, contemporary)
- [ ] **Maang tikka** and matha patti
- [ ] **Nath** (nose ring) and chain
- [ ] **Earrings** for each event
- [ ] **Necklace sets** — short and long (rani haar) for wedding
- [ ] **Choker** if desired
- [ ] **Bangles** — chooda (Punjabi), glass, gold, kaleerein
- [ ] **Haath phool** / hand harness
- [ ] **Rings** — engagement and wedding
- [ ] **Waist belt** (kamarbandh)
- [ ] **Anklets** (payal) and toe rings (bichhiya)
- [ ] **Armlets** (baajuband)
- [ ] Jewelry insurance
- [ ] Safe storage plan for event days

### Bride's Beauty
- [ ] Book makeup artist (MUA) for each event
- [ ] Makeup trial with MUA
- [ ] Book hair stylist
- [ ] Hair trial with stylist
- [ ] Decide on hairstyles per event (updo, braid, open, half-up)
- [ ] Book mehndi artist for bride's intricate hands and feet
- [ ] Plan skincare regimen (start 6+ months out)
- [ ] Plan facials timeline (last one 3–5 days before)
- [ ] Plan hair treatments (oiling, spa, keratin if desired)
- [ ] Plan threading / waxing / laser schedule
- [ ] Nail appointments (mani/pedi before mehndi)
- [ ] Teeth whitening if desired
- [ ] Gym / fitness routine
- [ ] Meal planning for skin and energy
- [ ] Mental wellness / therapy / meditation plan

### Groom's Wardrobe
- [ ] **Wedding sherwani** — color, embroidery, silhouette
- [ ] **Safa / pagdi** (turban) with kalgi brooch
- [ ] **Mojris / juttis** (shoes) — often embroidered
- [ ] **Dupatta / stole**
- [ ] **Sehra** (face veil of flowers) if tradition calls for it
- [ ] **Kirpan / sword** if tradition calls for it
- [ ] **Reception suit or indo-western**
- [ ] **Sangeet outfit** (bandhgala, nehru jacket, or playful kurta)
- [ ] **Mehndi kurta**
- [ ] **Haldi kurta** (designated to stain)
- [ ] **Welcome dinner outfit**
- [ ] Socks, pocket squares, cufflinks
- [ ] Watch for wedding day
- [ ] Grooming kit and backup

### Groom's Jewelry & Accessories
- [ ] Kalgi and brooch for safa
- [ ] Kada (bracelet)
- [ ] Rings
- [ ] Chain if desired
- [ ] Cufflinks and studs
- [ ] Walking stick if tradition includes it

### Family & Bridal Party Attire
- [ ] Coordinate parents' outfits (both mothers, both fathers)
- [ ] Coordinate siblings' outfits per event
- [ ] Decide on bridesmaids' outfits (coordinated or matching)
- [ ] Decide on groomsmen's outfits
- [ ] Arrange for outfits for grandparents
- [ ] Coordinate with cousins on sangeet performance outfits
- [ ] Order coordinated outfits for flower girls / ring bearer

### Guest Attire Style Guide
- [ ] Define dress code per event
- [ ] Create visual style guide (Pinterest board or PDF)
- [ ] Include color recommendations per event
- [ ] Include what NOT to wear (e.g., red for non-bride at wedding, black at some traditional events, white at certain ceremonies)
- [ ] Provide guidance for non-Indian guests (where to shop, how to drape a saree, kurta basics)
- [ ] Include links to recommended shops / online stores
- [ ] Include photos of appropriate looks per event
- [ ] Share style guide on wedding website
- [ ] Address weather and footwear (outdoor ceremonies, grass venues, removal for temple)
- [ ] Address modesty expectations if at religious venue
- [ ] Offer sari-draping service or video tutorials for non-Indian guests

## Phase 4: Vendors — Experience Layer (6–9 Months Before)

### Catering
- [ ] Define cuisine style per event (North Indian, South Indian, Indo-Chinese, Continental, fusion)
- [ ] Confirm vegetarian / Jain / vegan / halal requirements
- [ ] Research caterers with Indian wedding experience
- [ ] Schedule tastings with 2–3 finalists
- [ ] Book caterer
- [ ] Design menus per event:
  - [ ] Welcome dinner menu
  - [ ] Mehndi menu (light, often chaat-focused)
  - [ ] Haldi menu (traditional, often homestyle)
  - [ ] Sangeet menu (elaborate, multi-station)
  - [ ] Wedding lunch/dinner menu
  - [ ] Reception menu (often different cuisine for variety)
  - [ ] Post-wedding brunch menu
- [ ] Plan live stations (chaat, dosa, pasta, grill, paan, kulfi)
- [ ] Plan dessert spread (mithai, halwa, jalebi, western desserts)
- [ ] Plan welcome drinks and mocktails
- [ ] Plan signature cocktails
- [ ] Confirm head count per event
- [ ] Discuss dietary accommodations (allergies, gluten-free, diabetic)
- [ ] Plan vendor meals
- [ ] Plan kids' meal options

### Bar & Beverage
- [ ] Decide alcohol policy (open bar, limited, dry)
- [ ] Determine who provides alcohol (venue, caterer, self)
- [ ] Calculate quantity per event
- [ ] Stock wine, beer, liquor, mixers
- [ ] Design signature cocktail(s) with names tied to wedding brand
- [ ] Plan non-alcoholic options (virgin mojito, masala chai bar, coffee bar, lassi bar)
- [ ] Book bartenders
- [ ] Arrange glassware rentals
- [ ] Purchase bar liability insurance if required

### Decor & Florals
- [ ] Hire decorator/florist
- [ ] Share mood boards per event
- [ ] **Mandap design** — structure, florals, fabric, seating
- [ ] **Baraat decor** — entrance, horse/car decoration
- [ ] **Sangeet stage** — backdrop, seating, dance floor lighting
- [ ] **Mehndi setup** — low seating, cushions, swings, floral installations
- [ ] **Haldi setup** — marigolds, yellow drapes, flower petals
- [ ] **Reception stage** — backdrop, couple's seating
- [ ] **Entrance decor** for each event
- [ ] **Table centerpieces** per event
- [ ] **Welcome signage**
- [ ] **Escort cards / seating charts**
- [ ] **Photo booth backdrop**
- [ ] **Ceiling installations** (hanging florals, diyas, pom-poms)
- [ ] **Aisle decor** for ceremony
- [ ] **Pathway lighting / rangoli / diyas**
- [ ] Flower choices (marigold, rose, jasmine, orchid, lily)
- [ ] Confirm delivery and teardown logistics

### Lighting & Production
- [ ] Hire lighting designer or confirm via decorator
- [ ] Plan uplighting per venue
- [ ] Plan dance floor lighting for sangeet
- [ ] Plan fairy lights / string lights for outdoor events
- [ ] Plan pin-spotting for centerpieces
- [ ] Plan GOBO projection (monogram on dance floor)
- [ ] Rent sound system and microphones
- [ ] Book AV technician

### Music & Entertainment
- [ ] Book DJ for sangeet and reception
- [ ] Share music preferences and do-not-play list
- [ ] Book live musicians:
  - [ ] Dhol player for baraat
  - [ ] Shehnai / nadaswaram for ceremony
  - [ ] Classical vocalists / instrumentalists
  - [ ] Bollywood band for sangeet
  - [ ] Ghazal / qawwali / sufi singers for mehndi
- [ ] Book dance performers (garba troupe, bhangra, classical)
- [ ] Book emcee / MC (ideally bilingual)
- [ ] Plan sangeet performances schedule
- [ ] Book choreographer for family dances
- [ ] Schedule choreography sessions
- [ ] Plan grand entrance concept for couple
- [ ] Plan first dance song
- [ ] Plan parents' dances
- [ ] Book entertainment for cocktail hour (acoustic musicians, magician, caricaturist)
- [ ] Book fireworks / sparklers / cold pyro if permitted
- [ ] Book baraat extras (nachaniyas, flag bearers, fire twirlers, LED dhols)

### Transportation
- [ ] **Baraat vehicle** — horse, elephant, vintage car, convertible, Rolls Royce
- [ ] **Bride's entrance** — doli, palanquin, phoolon ki chaadar (floral canopy)
- [ ] Guest shuttles between hotel and venue for each event
- [ ] Airport pickup coordination for out-of-town guests
- [ ] VIP transport for elderly guests
- [ ] Parking attendants and valet
- [ ] Couple's getaway car after wedding
- [ ] Shuttle schedule and signage
- [ ] Driver list and contact sheet
- [ ] Backup vehicles

### Invitations

  - [ ] Event-specific inserts (mehndi, sangeet, haldi, wedding, reception)
  - [ ] RSVP card or website link
  - [ ] Accommodation insert
  - [ ] Travel / directions insert
  - [ ] Dress code insert
  - [ ] Kids policy insert
  - [ ] Registry insert (or website reference)
- [ ] Language decisions (English, Hindi, Sanskrit, regional scripts, bilingual)
- [ ] Include Ganesh motif or shloka at top (traditional)
- [ ] Include both sets of parents' names in proper order
- [ ] List of invited relatives ("with best compliments from...")

- [ ] Weigh invitation suite for correct postage

- [ ] **Favors tags / packaging**
- [ ] **Hashtag signs**

- [ ] Include a welcome note from the couple
- [ ] Include a small local specialty gift
- [ ] Include a copy of the hashtag/wedding brand card
- [ ] Customize bags with monogram
- [ ] Arrange delivery to hotel rooms or front desk distribution

## Phase 6: Guest Management (4–8 Months Before)

### Guest List
- [ ] Compile master guest list
- [ ] Categorize by side (bride's family, groom's family, friends, colleagues)
- [ ] Categorize by event (some guests are wedding-only, others all events)
- [ ] Set cap per side and negotiate
- [ ] Collect addresses
- [ ] Collect emails and phone numbers
- [ ] Collect dietary restrictions
- [ ] Collect accessibility needs
- [ ] Note VIPs (grandparents, elderly, pregnant, kids)
- [ ] Note "plus one" policies
- [ ] Note out-of-town vs local

### RSVP Tracking
- [ ] Set up RSVP system with per-event toggles
- [ ] Send RSVP reminders at T-minus 6 weeks, 4 weeks, 2 weeks
- [ ] Chase non-responders personally (call or text)
- [ ] Track attendance by event
- [ ] Share headcount updates with vendors weekly in final month

### Seating
- [ ] Build seating chart for mehndi (often floor seating + tables)
- [ ] Build seating chart for sangeet (round tables, reserved front)
- [ ] Build seating chart for wedding ceremony
- [ ] Build seating chart for reception
- [ ] Reserve front rows for immediate family at each event
- [ ] Reserve accessible seating
- [ ] Seat elderly away from speakers
- [ ] Seat kids together or with parents
- [ ] Plan single-friends table
- [ ] Plan non-Indian friends table with a "cultural host" (someone who can explain rituals)

### Guest Communication
- [ ] Set up guest WhatsApp group or email list
- [ ] Send pre-arrival info packet
- [ ] Send week-of reminder with detailed schedule
- [ ] Send day-of location and timing updates
- [ ] Plan post-wedding thank you communication

## Phase 7: Ceremony Specifics (3–6 Months Before)

### Wedding Ceremony Planning
- [ ] Meet with priest to walk through rituals in detail
- [ ] Confirm ceremony order:
  - [ ] Baraat arrival
  - [ ] Milni (family greeting)
  - [ ] Jaimala / varmala (garland exchange)
  - [ ] Ganesh puja
  - [ ] Kanyadaan
  - [ ] Mangal pheras (4 or 7 rounds)
  - [ ] Saptapadi (seven steps)
  - [ ] Sindoor / mangalsutra
  - [ ] Ashirwad (blessings)
  - [ ] Vidaai (farewell)
- [ ] Document muhurat timing for key moments
- [ ] Assign family roles (who does kanyadaan, who holds chadar)
- [ ] Procure puja samagri (ceremony items):
  - [ ] Kalash
  - [ ] Coconut
  - [ ] Rice, haldi, kumkum
  - [ ] Diya / oil lamp
  - [ ] Flowers and garlands
  - [ ] Ghee for agni
  - [ ] Wood for havan kund
  - [ ] Fruits and sweets for offerings
  - [ ] Mangalsutra
  - [ ] Sindoor box
  - [ ] Ring for ring-finding game
  - [ ] Silver/copper items per tradition
- [ ] Confirm who brings which items (priest, couple, decorator)
- [ ] Plan ceremony microphone and translation
- [ ] Plan family seating during ceremony
- [ ] Plan photography angles during ceremony
- [ ] Plan any games (joota chupai — shoe stealing, ring in milk)

### Mehndi Ceremony Planning
- [ ] Book mehndi artists (calculate: 1 artist per 15–20 guests)
- [ ] Bride's mehndi session (4–6 hours, done first)
- [ ] Plan bridal mehndi design and hidden names/initials
- [ ] Guest mehndi stations
- [ ] Plan ghodi / low seating / cushions
- [ ] Plan dhol and music
- [ ] Plan live singers (geet, tappe)
- [ ] Menu: finger foods, chaat, fresh juices
- [ ] Lemon wedges, sugar water, oil for mehndi setting
- [ ] Mehndi favors for guests

### Sangeet Planning
- [ ] Build run-of-show with emcee
- [ ] List and schedule performances:
  - [ ] Bride's family
  - [ ] Groom's family
  - [ ] Friends
  - [ ] Couple's couple dance
  - [ ] Kids' performance
- [ ] Coordinate rehearsals
- [ ] Prepare slideshow / video of couple's journey
- [ ] Plan speeches (parents, siblings, best man / maid of honor)
- [ ] Plan games (Mr. & Mrs., shoe game, trivia)
- [ ] Open dance floor plan

### Haldi Planning
- [ ] Plan haldi paste (turmeric, sandalwood, rose water, milk)
- [ ] Old/designated clothes for bride and groom
- [ ] Plastic sheets / towels for flooring protection
- [ ] Shower / rinse plan afterward
- [ ] Flower petal shower concept
- [ ] Music (traditional folk)
- [ ] Menu: traditional homestyle

### Reception Planning
- [ ] Plan grand entrance for couple
- [ ] First dance
- [ ] Cake cutting
- [ ] Speeches
- [ ] Toast / champagne pour
- [ ] Garter / bouquet toss (if fusion)
- [ ] Open dance floor
- [ ] Farewell / send-off concept
